## Setup Instructions

1. Clone the repository:
   ```sh
   git clone https://github.com/urmilgadhiya18/EduCREST.git
   ```
2. Navigate to the project directory:
   ```sh
   cd EduCREST
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm run dev
   ```
5. Open your browser and go to:
   ```sh
   http://localhost:3000
   ```
